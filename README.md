# 「お薬の説明」アプリ

漢方薬局ハレノヴァのお客さまがお薬を検索・印刷するためのWebアプリケーションです。

## 機能

- お薬の番号で検索（完全一致・部分一致対応）
- 複数キーワードを OR検索（スペースまたはカンマ区切り）
- A5サイズ印刷レイアウト自動生成
- スマホ・タブレット対応

## 使い方

1. お薬の番号を入力
2. 「検索する」ボタンをクリック
3. 検索結果が表示されます
4. 印刷が必要な場合は、自動表示される印刷レイアウトから印刷できます

## 技術スタック

- Python 3.9+
- Streamlit
- pandas
- openpyxl
